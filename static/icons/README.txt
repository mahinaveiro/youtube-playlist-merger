# PWA Icons Placeholder

This folder is the placeholder structure for your PWA icons. Please place the following images in this directory to complete the PWA integration without any extra work. They must be PNG format and exactly match these names and dimensions:

- `icon-72x72.png`    (72 x 72 pixels)
- `icon-96x96.png`    (96 x 96 pixels)
- `icon-128x128.png`  (128 x 128 pixels)
- `icon-144x144.png`  (144 x 144 pixels)
- `icon-152x152.png`  (152 x 152 pixels)   <-- Especially important for iOS (iPad)
- `icon-192x192.png`  (192 x 192 pixels)   <-- Primary Apple Touch Icon / Android icon
- `icon-384x384.png`  (384 x 384 pixels)
- `icon-512x512.png`  (512 x 512 pixels)   <-- Splash Screen / Large Android icon

Once these icons are placed in this folder, your application automatically links them via `index.html` and `manifest.json`.
